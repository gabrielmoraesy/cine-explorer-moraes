## 📋 Descrição do projeto
Avaliação técnica para o cargo de Estagiário
A aplicação foi desenvolvida em TypeScript, React e Next. utilizou-se NextJs para gerenciar as Rotas e API da tmdb.

- TypeScript
- React
- NextJs
